# MAN-TER SHOP V3

Proyecto final Front-End basado en un e-commerce industrial para MAN-TER GROUP.

## Funcionalidades

- HTML semántico.
- Bootstrap.
- CSS externo con Flexbox, Grid y Media Queries.
- Catálogo dinámico cargado con Fetch API desde `data/productos.json`.
- Cards de productos con animaciones.
- Buscador instantáneo.
- Favoritos visuales con LocalStorage.
- Carrito con LocalStorage.
- Contador visible en el menú.
- Carrito rápido tipo Offcanvas.
- Página `carrito.html` con edición de cantidades, eliminación, subtotal, IVA y total.
- Formulario validado con JavaScript y preparado para Formspree.
- Modo oscuro persistente.
- SEO básico y accesibilidad.
- Galería multimedia con Slick Carousel.
